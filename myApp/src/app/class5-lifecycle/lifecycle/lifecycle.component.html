Angular Lifecycle

  1. ngOnChanges - called after a input(@Input) property changes
  2. ngOnInit - called once the component is initialized, not visible
  3. ngOnDoCheck - it runs lots time, whenever the angular runs change detection, like value cahnges, dom change, button click
  4. ngAfterContentInit - called every time after content (ng-content) projected in view
  5. ngAfterContentChecked - called every time (ng-content) projected in view
  6. ngAfterViewInit - called once the view of the component is initialized
  7. ngOnViewChecked - called every time the view has any changes
  8. ngOnDestroy - called when the component is about to destroy
